# Go 短视频 Feed 流系统 - 30节课程大纲

> 本课程基于 `feedsystem_video_go` 项目，带你从零学习 Go Web 开发实战

---

## 课程目标

- 掌握 Go Web 开发完整流程
- 深入理解 Gin + GORM + Redis + RabbitMQ 实战应用
- 学习高并发系统设计思想（缓存、异步、队列）
- 具备独立开发 RESTful API 的能力

---

## 课程大纲

### 基础入门篇（第1-8节）

| 课程 | 主题 | 内容 |
|------|------|------|
| 第1节 | 项目概览与环境 | 了解项目整体架构、技术栈、启动方式 |

## 学习建议

### 阶段一：基础入门（1-8节）
- 先跑通项目，理解整体架构
- 重点掌握 Go 基础语法和 Gin 路由

### 阶段二：核心业务（9-16节）
- 从账号模块入手，理解分层架构
- 逐个模块学习 CRUD 实现

### 阶段三：Feed 流（17-22节）
- 理解不同 Feed 流的设计思路
- 重点掌握分页游标设计

### 阶段四：缓存与异步（23-30节）
- 这是项目的亮点和难点
- 需要反复实践和思考
